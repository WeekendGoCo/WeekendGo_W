# وصف الشعار (Logo Description)

يتكون الشعار من:
1. حقيبة سفر حديثة باللون الأزرق.
2. خطوط انسيابية ديناميكية بالأزرق والأخضر تمثل الحركة والسفر.
3. سهم متجه للأعلى يرمز للتقدم والانطلاق.
4. كتابة عربية "ويكند جو".
5. كتابة إنجليزية "WEEKEND GO".
6. سطر فرعي: "للسفر والسياحة".

## رمزية التصميم
- الأزرق = الثقة والاحتراف.
- الأخضر = الراحة والطاقة والحيوية.
- الخطوط المنحنية = الحركة والسلاسة.
- السهم = الطموح والانطلاق.
